Painting Grid

Overview 
Memorise the colours present in the shown grid within a time limit and paint the squares in the grid within another given time limit. For some game mode, there are 3 levels: easy, medium and hard. First-time users must accomplish the easy level first then only able to unlock the medium level, same for hard level. When all levels are unlocked, the user can challenge whatever level the user likes. The points earned can be used to redeem cash vouchers to buy the product. 

Game mechanics
Common game features
Winning threshold: At least 50% of the grid is correctly painted, including blanks 
-	Easy: 5
-	Medium: 8
-	Hard: 18

Grid size
-	Easy: 3x3
-	Medium: 4x4
-	Hard: 6x6

Points when the level is passed
-	Easy: 20
-	Medium: 50
-	Hard: 80

Bonus Points (exclude Easy level)
Action 	                            Bonus 
100% accuracy	                    +50
Finish with ≥ 20% time left	        +20
≥ 4 perfect rounds in a row 	    +100 with +40 when maintaining the streak after winning 4 rounds


Given time limit 
These are for 3 colours, 20 seconds is added when a new colour involved
Memorise time limit (seconds)
-	Easy & Medium: 5
-	Hard: 10

Painting time limit (seconds)
-	Easy: 30
-	Medium: 50
-	Hard: 90

Tile colours
-	Red 
-	Orange 
-	Yellow
-	Green 
-	Blue 
-	Purple 
-	Pink 

All colours have light, bright & dark
 
Mode 1: Classic mode
Show the full grid
User paints the tiles based on memory
Pass the level if reached the winning threshold
*Points and winning threshold is stated in the common game features

Mode 2: Reverse memory mode
Show the full grid 
Player receives one instruction
Paint the tiles according to the instruction

Instructions:
-	Paint only 1 colour tiles
-	Paint some colours tiles
-	Paint everything except a colour 
-	Paint light/bright/dark colour tiles

Winning threshold: correctly painter 50% of the relevant tiles according to the instruction

Time limit – painting 
-	Easy: 15
-	Medium: 30 
-	Hard: 50

Points
Action 	Points 
Correct target tile painted	+10
Target tile missed	-5
Incorrect tile painted	-8
Correctly ignored tiles	+2

Mode 3: Survival mode
Start at Easy, then progress to Medium and then Hard
After each successful round, the next round either:
-	Grid size increase
-	Number of colours increase
-	Memorise/painting time decrease
Player continues until:
-	Time runs out 
OR
-	Accuracy drops below threshold

Points system
Points = correct tiles x difficulty multiplier 
Difficulty multiplier
-	Easy: x1
-	Medium: x1.5
-	Hard: x2
